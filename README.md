# Cheatsheet — Statistical Modelling (401-3622-00L)

Cheatsheet for the course **Statistical Modelling** at ETH Zürich, taught by Prof. Peter Bühlmann (Autumn Semester 2025).

## Topics Covered

- Simple and multiple linear regression
- General linear models
- High-dimensional models
- Non-linear models and generalized linear models (GLMs)
- Robust regression methods
- Model selection and residual analysis
- Non-parametric models
